Architecture & Design Patterns

// MVVM Architecture Implementation
public class AlarmViewModel extends AndroidViewModel {
    private MutableLiveData<List<Alarm>> alarms;
    private AlarmRepository repository;
    
    public AlarmViewModel(@NonNull Application application) {
        super(application);
        repository = new AlarmRepository(application);
        alarms = repository.getAllAlarms();
    }
    
    public LiveData<List<Alarm>> getAllAlarms() {
        return alarms;
    }
    
    public void insert(Alarm alarm) {
        repository.insert(alarm);
    }
}

Firebase Integration

// Firebase Authentication & Firestore
public class FirestoreManager {
    private FirebaseFirestore db;
    private FirebaseAuth auth;
    
    public void saveUserProfile(UserProfile profile) {
        db.collection("users")
          .document(auth.getCurrentUser().getUid())
          .set(profile)
          .addOnSuccessListener(aVoid -> {
              Log.d(TAG, "Profile saved successfully");
          });
    }
    
    public void uploadProfileImage(Uri imageUri) {
        StorageReference storageRef = FirebaseStorage.getInstance()
            .getReference("profile_images/" + auth.getCurrentUser().getUid());
        
        storageRef.putFile(imageUri)
            .addOnSuccessListener(taskSnapshot -> {
                // Handle successful upload
            });
    }
}

App Versions

Main Branch (Full-Featured Version)

  • Firebase Integration: Complete authentication, Firestore database, and Storage
  • Google Maps: Location-based features and time zone calculations
  • User Management: Profile creation, photo uploads, and data synchronization
  • Background Services: Reliable alarm notifications and timer management

Alternative Branch (Simplified Version)

  • Local-Only: No Firebase or external API dependencies
  • Basic Functionality: Core time management features without authentication
  • Offline Operation: Works without internet connection
  • Lightweight: Reduced app size and complexity

Key Components

  • Activities: LoginActivity, RegisterActivity, MainActivity, MapActivity
  • Fragments: AlarmFragment, TimerFragment, StopwatchFragment, WorldClockFragment
  • Services: AlarmService, TimerService, WorldClockWidget
  • Managers: FirestoreManager, TimeZoneManager, PreferencesManager

Development Process
