Research Overview

Developed automated pipelines for analyzing coarse-grained DNA assembly simulations using LAMMPS molecular dynamics. This project focused on understanding the thermodynamic and rheological properties of self-assembling DNA nanostructures under different interaction parameters.

Key Research Areas

Rheological Analysis

Implemented Green-Kubo relations for calculating storage (G') and loss (G'') moduli from stress autocorrelation functions, enabling characterization of viscoelastic properties.

Parameter Optimization

Systematic study of interaction strength (Îµ) and angular rigidity (k) parameters across multiple simulation conditions to understand assembly behavior.

Statistical Validation

Comprehensive statistical analysis with bootstrap confidence intervals and cross-validation to ensure robust scientific conclusions.

Structural Analysis

Cluster analysis and radial distribution function calculations to understand the relationship between molecular structure and material properties.

Methodology Overview

1. Simulation Setup

  • Coarse-grained bead-spring model for DNA building blocks
  • Lennard-Jones interactions with tunable parameters
  • Harmonic angle potentials for structural flexibility

2. Data Processing Pipeline

  • Automated SACF (Stress Autocorrelation Function) analysis
  • Green-Kubo integration for rheological properties
  • Statistical validation with bootstrap methods

3. Visualization & Analysis

  • Publication-ready scientific plots
  • Parameter space exploration
  • Cross-validation and error analysis
